# lkr-fluctuation-against-usd
A data-driven analysis using csbl data

## Overview
This study aims to visualize the volatility of lkr against usd and forecast next 90 days, analyze the macroeconomic behavior of lkr against usd during recent time, I identified key three phases in recent year such as the crisis period (2022-2023), post-crisis stabilization (2024-mid 2025) and current trend (mid 2025-present).

## Technical platforms
Pandas, Matplotlib, seaborn, statsmodels, jupyter notebook
